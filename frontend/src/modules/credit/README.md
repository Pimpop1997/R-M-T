

## New UI Component: Borrow Credit Form

The Borrow Credit Form allows users to request a credit loan by entering the desired amount of credits. 

### Usage Example

```jsx
import BorrowCreditForm from './BorrowCreditForm';

function App() {
  return (
    <div className="App">
      <BorrowCreditForm />
    </div>
  );
}
```
